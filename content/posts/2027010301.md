**A:** Yes, sharing your public referral link is safe and encouraged by Binance. However, never share your referral *dashboard* login details or any private account information.

**Q: What's the difference between a referral code and an invitation link?**
**A:** They function identically. The invitation link (`binance.com/en/join?ref=CODE`) simply has the code embedded within it. Using the link ensures the code is pre-filled and reduces user error.

**Q: How long do the referral fee discounts last?**
**A:** For the referred user, the trading fee discount is typically permanent for as long as the account is in good standing. The specific discount rate (e.g., 20%) is fixed at registration.
